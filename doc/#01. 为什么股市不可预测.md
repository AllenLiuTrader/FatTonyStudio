# 为什么股市不可预测

`2020-04-05`

## 相关信息不完整

要做出判断，首先要有信息。大多数人只能根据公开信息进行计算，比如财报；少部分知情人能获得更多相关信息，比如上市公司的高管。但是没有人知道全部信息。一方面，社会是一个巨大的系统，一家上市公司并非封闭的个体，而是社会系统的一部分。它可能受到公司外部针对性的攻击，比如竞争对手的挤兑甚至陷害；也可能受到其他力量无意的波及，比如台风地震等天灾干扰了消费市场。另一方面，上市公司作为一个系统，对任何人都会有被忽略掉的盲区。基层员工不知道高层决策，高管不知道基层的痛点问题。所有这一切，都导致用来做出判断的信息集合是不完整的。

## 事件关系不确定

即使有完整信息，也难以进行准确判断。首先，世界上有很多事件，这些事件之间的因果关系是多对多的，而非线形孤立的。基本不可能说事件A导致了事件B，世界没有那么简单。实际上更可能是A组事件导致了B组事件，而我们关注的结果凑巧在B组。更麻烦的是，B组事件会继续作用，衍生出C组事件。世界的复杂度其实非常惊人，根本不是线性思维可以理解的，它至少是一个多维的网状结构。这个网络的每个节点是一个事件，而节点之间的线则是因果关系。此外，这个世界不只有因果关系，还有随机性。也就是没有为什么，只是凑巧产生了一个结果而已。我们在回顾历史时总是能找出一堆原因，给出一个看起来很有道理的解释。其实那是马后炮，处在当时的状态几乎预测不到那样的结果。最后，有一种东西叫做黑天鹅事件。极其稀有，毫无征兆，影响巨大。即便你找到所有因果关系，并且覆盖了所有可能的随机结果，仍然会被黑天鹅一击致命。比如这次的中共肺炎，没人预测到，却给全人类剥了一层皮。

```
相关信息不完整，且不可能完整；事件关系不确定，且不可能确定。因此，股票市场是无法准确预测的。
```
